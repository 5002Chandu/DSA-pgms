# Data Structures and Algorithms (DSA) Programs

This repository contains basic **Data Structures and Algorithms (DSA)** programs implemented for learning and practice purposes. These programs help in understanding fundamental concepts used in problem solving and software development.

## 📚 Topics Covered

* Arrays
* Linked List (Singly & Doubly)
* Stack
* Queue
* Circular Queue
* Searching Algorithms
* Sorting Algorithms
* Recursion
* Hashing Basics

## 💻 Programming Language

* C / C++ (Basic implementation for academic practice)

## 🎯 Purpose

The main goal of this repository is to:

* Practice DSA concepts
* Improve programming logic
* Prepare for coding interviews and exams

## ▶️ How to Run

1. Clone the repository
2. Open the program in any C/C++ compiler
3. Compile and run the program

Example:

```
gcc filename.c
./a.out
```
